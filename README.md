# B3 project repo list

List of the software repos developed in the B3 project.
https://b-cubed.eu/

---

### [https://github.com/mtrekels/b3alien](https://github.com/mtrekels/b3alien)

**b3alien** – Tools and workflows related to alien (non-native) species data processing and quality assessment within the B-CUBED framework.

---

### [https://github.com/b-cubed-eu/b3data-scripts](https://github.com/b-cubed-eu/b3data-scripts)

**b3data-scripts** – Utility scripts for data ingestion, transformation, validation, and automation used across B-CUBED datasets and pipelines.

---

### [https://github.com/b-cubed-eu/b3doc](https://github.com/b-cubed-eu/b3doc)

**b3doc** – Central documentation repository for B-CUBED software, standards, methodologies, and quality indicators.

---

### [https://github.com/b-cubed-eu/b3gbi](https://github.com/b-cubed-eu/b3gbi)

**b3gbi** – Backend services and logic for B-CUBED’s Global Biodiversity Indicators, including computation and integration workflows.

---

### [https://github.com/b-cubed-eu/b3gbi-gui](https://github.com/b-cubed-eu/b3gbi-gui)

**b3gbi-gui** – Web-based graphical user interface for exploring, configuring, and visualizing B-CUBED Global Biodiversity Indicators.

---

### [https://github.com/AgentschapPlantentuinMeise/binfrastructure](https://github.com/AgentschapPlantentuinMeise/binfrastructure)

**binfrastructure** – Infrastructure-as-code and deployment tooling maintained by Meise Botanic Garden for biodiversity-related services.

---

### [https://github.com/b-cubed-eu/comp-unstructured-data](https://github.com/b-cubed-eu/comp-unstructured-data)

**comp-unstructured-data** – Components for comparing, processing, and assessing quality of unstructured biodiversity data sources.

---

### [https://github.com/b-cubed-eu/dissmapr](https://github.com/b-cubed-eu/dissmapr)

**dissmapr** – R-based tools for generating and analysing species distribution and dissemination maps.

---

### [https://github.com/trias-project/indicators](https://github.com/trias-project/indicators)

**indicators** – Implementation of biodiversity and invasion-related indicators developed within the TRIAS project.

---

### [https://github.com/b-cubed-eu/integrate-indicator-software](https://github.com/b-cubed-eu/integrate-indicator-software)

**integrate-indicator-software** – Glue code and workflows to integrate indicator software into broader B-CUBED data pipelines.

---

### [https://github.com/b-cubed-eu/invasimapr](https://github.com/b-cubed-eu/invasimapr)

**invasimapr** – R package for mapping, analysing, and visualising invasive species distributions.

---

### [https://github.com/gbif/occurrence-cube](https://github.com/gbif/occurrence-cube)

**occurrence-cube** – GBIF project implementing data cubes for aggregated biodiversity occurrence data.

---

### [https://github.com/b-cubed-eu/pdindicatoR](https://github.com/b-cubed-eu/pdindicatoR)

**pdindicatoR** – R package for computing population and pressure indicators used in biodiversity assessments.

---

### [https://github.com/gbif/pygbif](https://github.com/gbif/pygbif)

**pygbif** – Official Python client for accessing GBIF APIs and biodiversity occurrence data.

---

### [https://github.com/ropensci/rgbif](https://github.com/ropensci/rgbif)

**rgbif** – R client for the GBIF API, widely used for biodiversity data retrieval and analysis.

---

### [https://github.com/b-cubed-eu/rsa-unstructured-data-comp](https://github.com/b-cubed-eu/rsa-unstructured-data-comp)

**rsa-unstructured-data-comp** – Components for risk or sensitivity assessment of unstructured biodiversity data.

---

### [https://github.com/b-cubed-eu/virtual-suitability-cube](https://github.com/b-cubed-eu/virtual-suitability-cube)

**virtual-suitability-cube** – Data cube implementation for modelling and analysing species habitat suitability.

---

### [https://github.com/trias-project/trias](https://github.com/trias-project/trias)

**trias** – Core repository of the TRIAS project, focused on tracking and analysing invasive alien species in Belgium.
